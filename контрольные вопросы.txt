Репозиторий — это хранилище файлов проекта, где хранится история изменений, версии и ветки разработки.
Публичный репозиторий доступен всем пользователям, тогда как приватный доступен только определённым людям или командам.
Файл README содержит описание проекта, инструкции по установке и использованию, а также другую важную информацию для пользователей и разработчиков.
Файл .gitignore определяет файлы и папки, которые нужно игнорировать при коммитах (например, временные файлы или файлы конфигурации).
Команда git add добавляет изменения в индекс Git перед коммитом.
Команда git commit фиксирует внесённые изменения в репозиторий с указанием комментария.
Команда git push отправляет локальные коммиты в удалённый репозиторий.
